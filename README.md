# 🚀 VPS 聚合优化面板

## 一、 📖 项目简介
这是一个集**环境初始化、系统调优、网络加速、安全加固、探针测速**及**全能节点建站**于一体的 VPS 终极 Bash 脚本。
专为折腾 VPS 的玩家打造，帮你省去到处找脚本的烦恼，**全局快捷键一键唤出，告别繁琐，直击痛点**！

## 二、 ✨ 核心功能矩阵

👉 **基础与系统环境**
* ✅ **一键初始化**：自动安装必备工具、校准时区限制日志大小、开启原生 BBR。
* ✅ **系统高级开关**：纯交互式傻瓜管理 IPv6、被 Ping 状态、自动更新、防火墙放行及系统深层垃圾清理。
* ✅ **全能软件库**：一键拉取 Docker、Python、宝塔面板、哪吒探针、WARP、Aria2 以及 Caddy 反代等常用环境。

👉 **安全与网络优化**
* ✅ **SSH 安全加固**：自定义 SSH 端口，自动配置多重防火墙（ufw/firewalld/iptables）保险放行，防爆破防失联。
* ✅ **Docker 防穿透**：一键限制 Docker 暴露端口仅允许本地 (`127.0.0.1`) 访问，防止服务被打。
* ✅ **极致网络调优**：内置 `tcpx.sh` 终极 BBR 加速合集，并支持 Omnitt 动态 TCP 防呆参数生成与挂载。

👉 **内核与硬件榨取**
* ✅ **智能内存优化**：根据机器物理内存大小，自动推荐激进、积极或保守的 ZRAM 压缩率与 Swap 策略。
* ✅ **内核管理**：Debian/Ubuntu 专属 Cloud 内核一键换装，清理冗余旧内核，释放 `/boot` 空间与硬件潜能。

👉 **跑分测速与建站探针**
* ✅ **神级测试合集**：内置 YABS 性能、融合怪终极测速、流媒体解锁检测、三网回程路由、欺诈 IP 检测等顶级测试库。
* ✅ **主流面板部署**：快速拉取 Sing-box（甬哥脚本）、x-panel 等多协议面板。

## 三、 💻 安装与使用指南

跟着下面的步骤，闭着眼睛直接跑：

**第一步：连接服务器**
确保你以 `root` 用户登录 VPS，如果不是，请先执行 `sudo -i` 切换为超级管理员。

**第二步：运行一键安装脚本**
```bash
wget -qO vps.sh https://raw.githubusercontent.com/Chunlion/VPS-Optimize/main/vps.sh && chmod +x vps.sh && ./vps.sh
```

**第三步：使用快捷键**
首次运行后，脚本会自动注册全局快捷指令。以后只需在终端敲入：
```bash
cy
```
即可随时随地唤出控制面板！⚡️

## 四、 💡 避坑与注意事项

🔴 **端口冲突预警（核心避坑）：**
如果你安装了 `x-ui` / `x-panel` 等节点面板，**并且使用了面板自带的 SSL 证书申请功能，这将会与本脚本中的「Caddy 一键反向代理」产生严重冲突！**（两者都会抢占服务器的 `80` 端口）。
👉 **解决方案**：证书申请必须**二选一**！
1. 要么完全依靠节点面板内部的证书申请逻辑。
2. 要么关闭面板的申请功能，把域名解析和 HTTPS 加密完全交由 Caddy 反代来统一保护。

🔴 **内核清理风险：**
执行面板中的「10. 卸载冗余旧内核」功能时，**千万不要卸载当前正在运行的内核及 Cloud 内核**，否则重启后服务器直接失联，神仙难救。

🔴 **云厂商防火墙警告：**
如果你使用的是阿里云、腾讯云、甲骨文等带有“安全组”控制台的 VPS，在使用面板修改 SSH 端口后，**必须手动在网页端开启新端口**，否则必失联。
