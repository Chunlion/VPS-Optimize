# 🚀 VPS-Optimize：VPS 优化脚本

 

>
> 这是一个专为 **中转节点、科学上网、高并发 Web 服务** 打造的一键优化工具。它深度整合了专业的 TCP 调优参数、BBR 拥塞控制算法以及系统安全加固逻辑，旨在解决跨境网络高延迟、丢包及系统连接数限制等痛点。

-----

## 🌟 核心深度优化项

### ⚡ 极致性能 (Network & Kernel)

  * **BBR 全速开启**：自动检测并启用内核级 BBR 拥塞控制算法，显著提升吞吐量。
  * **  TCP 调优**：
      * 👉 **缓冲区翻倍**：将 TCP 发送/接收缓冲区提升至 **32MB**，解决长肥管道（LFN）性能瓶颈。
      * 👉 **MTU 智能探测**：开启 `tcp_mtu_probing`，有效解决分片导致的玄学断连。
      * 👉 **连接复用**：优化 `TIME_WAIT` 回收与 `Fast Open`，减少握手延迟。
  * **句柄数爆发**：暴力提升系统文件描述符限制至 **1,000,000**，支持超高并发连接。

### 🛡️ 安全加固 (Security)

  * **SSH 端口防护**：支持交互式修改 SSH 端口，**回车即跳过**，保持现状。
  * **Fail2ban 联动**：自动配置暴力破解防护，10分钟内失败5次封禁24小时。
  * **防火墙自适应**：修改端口时自动尝试放行 `UFW` 或 `Iptables` 规则，防止失联。

### 🛠️ 智能环境 (Environment)

  * **Swap 自动配置**：针对小内存 VPS（\<2GB）自动创建 **2GB 虚拟内存**。
  * **时区对齐**：统一设置系统时区为 `Asia/Shanghai`。
  * **兼容性保障**：采用逐行应用 `sysctl` 技术，完美兼容 **KVM / OpenVZ / LXC** 架构。

-----

## 🚀 一键执行命令

在你的 VPS 终端（Root 用户）复制并粘贴以下命令：

```bash
wget -qO vps.sh https://raw.githubusercontent.com/Chunlion/VPS-Optimize/main/vps.sh && chmod +x vps.sh && ./vps.sh
```

> **💡 提示**：如果 GitHub 原始链接在国内访问困难，建议先通过代理或使用本地下载上传的方式执行。

-----

## 📖 操作步骤说明

1.  **执行脚本**：脚本会自动进行环境检测，并询问是否创建 Swap（如果内存小）。
2.  **网络加速**：自动写入内核优化参数，立即开启 BBR 算法。
3.  **SSH 交互**：
      * 脚本会显示当前 SSH 端口。
      * **如果你想修改**：输入新端口号并按回车。
      * **如果你不想修改**：**直接按回车**，脚本会跳过此步骤，确保安全。
4.  **安全防护**：安装并配置 Fail2ban，加固系统登录安全。
5.  **重启生效**：建议在脚本最后提示时选择 `y` 重启系统，让所有内核参数完美生效。

-----

## 🔴 避坑指南 & 注意事项

  * **防火墙警告**：如果你使用的是阿里云、腾讯云、甲骨文等带有“安全组”控制台的 VPS，**必须手动在网页端开启新 SSH 端口**。
  * **失联预防**：在修改端口并重启后，**请勿关闭当前窗口**，先开一个新终端尝试连接。
  * **内核要求**：建议使用 **Debian 10+ / Ubuntu 18.04+ / CentOS 7+** 等现代内核版本，以获得最佳 BBR 体验。
