# 菜单地图

这份菜单地图用于配合教程阅读。文档里的菜单路径不再写裸数字，也不再把快捷命令放进每条路径，而是写成：

```text
主菜单 [19 443 单入口管理中心] -> [3 443 单入口链路体检]
```

如果你刚登录服务器，需要先输入快捷命令进入主菜单；进入后按下面的编号和文案选择即可。

## 主菜单

| 编号 | 菜单文案 | 适合场景 |
|---|---|---|
| `n` | 新手向导 | 第一次使用，只看核心路径 |
| `1` | 运维预检与风险扫描 | 部署前检查系统、网络、端口、资源和风险 |
| `2` | 基础环境初始化 | 安装常用工具、设置时区、系统更新、基础 BBR |
| `3` | 基础组件与反代分流 | Docker、Caddy、WARP、普通反代、443 入口 |
| `4` | 面板、节点与订阅工具 | 3x-ui、S-UI、Sing-box、Xray、订阅工具、Dockge |
| `5` | SSH 安全加固 | 修改 SSH 端口、防失联、安全登录 |
| `6` | 添加 SSH 公钥 | 添加免密登录公钥 |
| `7` | Fail2ban 防爆破 | 防 SSH 爆破 |
| `8` | 防火墙规则管理 | 放行、删除、查看或关闭系统防火墙规则 |
| `9` | 系统开关与清理 | IPv6、IPv4 优先、Ping、自动更新、垃圾清理 |
| `10` | 网络与内核优化 | BBR、TCP 参数、ZRAM、轻量内核 |
| `11` | Docker 安全管理 | Docker 本地防穿透和恢复访问 |
| `12` | 测速与质量检测 | YABS、流媒体、回程、IP 质量 |
| `13` | 端口排查与释放 | 查看端口占用，必要时释放端口 |
| `14` | 系统硬件探针 | CPU、内存、磁盘、网络实时信息 |
| `15` | 服务健康总览 | 服务状态、证书摘要、端口概览、反馈诊断 |
| `16` | 配置备份与回滚 | 创建备份、查看备份、恢复配置、隔离旧备份 |
| `17` | 更新脚本 | 拉取最新脚本并刷新快捷入口 |
| `18` | 重启服务器 | 重启 VPS |
| `19` | 443 单入口管理中心 | 初始化、加网站、体检、证书修复 |

## 新手向导

| 路径 | 用途 |
|---|---|
| `主菜单 [n 新手向导] -> [1 新机器初始化]` | 预检、初始化、SSH、公钥、Fail2ban、防火墙、备份 |
| `主菜单 [n 新手向导] -> [2 安装面板/节点]` | 进入面板、节点与订阅工具菜单 |
| `主菜单 [n 新手向导] -> [3 配置 443 单入口]` | 面板、订阅、REALITY 共用公网 443 |
| `主菜单 [n 新手向导] -> [4 健康检查]` | 服务状态、端口、证书、反馈诊断 |
| `主菜单 [n 新手向导] -> [5 备份/回滚]` | 创建备份或恢复配置 |

主菜单 [3 基础组件与反代分流]

| `4` | Realm 端口转发 | 安装或管理 Realm |
| `5` | Gost 隧道 | 安装或管理 Gost |
| `6` | 极光面板 | 安装极光面板 |
| `7` | 哪吒监控 | 安装哪吒监控 |
| `8` | WARP 解锁/网络 | WARP 相关网络能力 |
| `9` | Aria2 下载 | 安装 Aria2 |
| `10` | 宝塔面板 | 安装宝塔面板 |
| `11` | PVE 虚拟化工具 | PVE 相关工具 |
| `12` | Argox 节点 | Argox 节点相关入口 |
| `13` | 普通 Caddy 反代 | 普通网站或面板反代，不是 443 单入口 |
| `14` | 查看 Caddy 证书路径 | 查看证书和私钥位置 |
| `15` | Caddy 跳过后端证书校验 | 后端是自签 HTTPS 时使用 |
| `16` | 清空 Caddy 配置 | 危险操作，会清理反代配置 |
| `17` | 删除底层 ACME 证书 | 危险操作，会清理签发记录 |
| `18` | 首次配置 443 单入口 | 旧入口，推荐优先走 `主菜单 [19 443 单入口管理中心] -> [1 首次配置 443 单入口]` |
| `19` | 443 证书与维护中心 | 旧入口，推荐优先走 `主菜单 [19 443 单入口管理中心] -> [6 CF DNS / Caddy 证书维护]` |
| `20` | 管理 443 网站/反代 | 旧入口，推荐优先走 `主菜单 [19 443 单入口管理中心] -> [2 管理网站/反代域名]` |

常见例子：

```text
主菜单 [3 基础组件与反代分流] -> [1 Docker 引擎]
主菜单 [3 基础组件与反代分流] -> [13 普通 Caddy 反代]
```

## 443 单入口管理中心

路径前缀：

```text
主菜单 [19 443 单入口管理中心]
```

| 子编号 | 菜单文案 | 用途 |
|---|---|---|
| `1` | 首次配置 443 单入口 | 第一次部署 443 单入口时使用 |
| `2` | 管理网站/反代域名 | 新增、删除、查看网站，最常用 |
| `3` | 443 单入口链路体检 | 检查 Nginx、Caddy、REALITY、面板、安全项 |
| `4` | 重新应用上次配置 | 读取 `sni-stack.env` 重新生成配置 |
| `5` | 订阅链接 / External Proxy 提示 | 检查节点链接是否输出公网 443 |
| `6` | CF DNS / Caddy 证书维护 | 重签、软链、清理、修复、回滚 |
| `7` | 修改 443 分流参数 | 面板、订阅、REALITY、入口端口与路径 |

常见例子：

```text
主菜单 [19 443 单入口管理中心] -> [1 首次配置 443 单入口]
主菜单 [19 443 单入口管理中心] -> [2 管理网站/反代域名]
主菜单 [19 443 单入口管理中心] -> [3 443 单入口链路体检]
```

```text
主菜单 [19 443 单入口管理中心] -> [7 修改 443 分流参数]
```

|---|---|---|
| `1` | 修改面板/订阅端口与路径 | 面板端口、订阅端口、路径前缀 |
| `2` | 修改 REALITY 本地监听 / 伪装 SNI | REALITY 本地端口和外部伪装 SNI |

| `4` | 修改面板域名 | 更换面板域名 |
| `5` | 重新应用当前保存的配置 | 读取当前配置并重建 Nginx/Caddy |

| `7` | 查看已管理域名 / 证书路径 | 查看 Caddy 域名和证书软链接 |
