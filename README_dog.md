👉 **一、 端口流量狗 v1.2.6 交互增强版：你的专属节点流量管家**

经常和朋友合租节点，或者给群友分享端口，但总有人跑满流量不知道？
**端口流量狗 (Port Traffic Dog)** 是一只轻巧、硬核的 Linux 端口流量守护犬，底层基于 `nftables` 和 `tc` 构建，占用极低，统计贼准。

> 💡 **致谢说明**：本工具的核心逻辑基于原作者开源项目 [zywe03/realm-xwPF](https://github.com/zywe03/realm-xwPF/blob/main/port-traffic-dog.sh) 进行增强与维护，感谢原作者的优秀底座！

🌟 **本次 v1.2.6 增强版核心升级：**
除了原有的**流量限额断网**、**带宽限速**、**单双向计费**外，本次加入了**【Telegram 群聊交互式查询】**功能！
群友不需要天天问你用了多少流量，直接在群里发送 `/t 端口号`，机器人秒回实时流量数据，服务器后台 Systemd 守护进程保活，绝不漏接任何一次查询。

👉 **二、 核心功能亮点**

✅ **双管齐下**：支持**流量总量配额**（超量自动断开连接）和**带宽速率限制**（精准限速不卡顿）。
✅ **计费灵活**：支持**双向计费**（上行+下行）或**单向计费**（仅算出站）。
✅ **自动重置**：可设定每月固定日期（如每月1号）自动清零流量，彻底解放双手。
🔴 **实时群聊交互**：内建常驻后台的机器人监听服务，直接读取网卡底层数据，群内查流量零延迟。

👉 **三、 新手保姆级部署教程**

**步骤 1：运行脚本**
```bash
wget -qO dog.sh https://raw.githubusercontent.com/Chunlion/VPS-Optimize/main/dog.sh && chmod +x dog.sh && ./dog.sh
```
> 💡 *提示：安装完成后，以后随时在终端输入 `dog` 即可唤出主菜单。*

**步骤 2：获取 TG 机器人 Token**
1. 去 Telegram 搜索 `@BotFather`，输入 `/newbot` 创建一个你的机器人。
2. 创建成功后，复制那段长长的 **API Token**。

**步骤 3：关闭机器人的隐私模式 (🔴 绝对关键的避坑点)**
默认情况下机器人是个“瞎子”，看不到群里普通人发的消息。
1. 在 `@BotFather` 聊天框输入 `/mybots`，选中你刚创建的机器人。
2. 依次点击 **Bot Settings** 👉 **Group Privacy**。
3. 点击 **Turn OFF**（确保屏幕上显示的是 Privacy mode is disabled）。

**步骤 4：在脚本中一键部署服务**
1. 在服务器的 `dog` 主菜单中，输入 `7` 选择 **通知管理**。
2. 输入 `1` 选择 **部署 Telegram 交互式查询机器人**。
3. 把刚才申请的 **Bot Token** 粘贴进去。
4. 输入你的 **Chat ID**（你想让机器人在哪个群工作，就填那个群的 ID，通常是以 `-100` 开头的数字）。

✅ **大功告成！** 脚本会自动帮你写好配置，并启动一个永远不死的后台服务。

👉 **四、 群友该怎么查流量？**

直接把你刚刚配置好的机器人拉进群里（最好给个管理员权限），然后任何人只要在群里发送指令：

* ` /t 8080 ` （查询 8080 端口）
* ` /traffic 10000-10050 ` （查询端口段）

机器人就会立刻回复：
> 📊 **端口 8080 实时流量查询**
> ────────────────
> ↑ 上行(入站): **1.20 GB**
> ↓ 下行(出站): **500.00 MB**
> 💰 累计总计: **1.69 GB**
> 📌 计费模式: 双向计费
